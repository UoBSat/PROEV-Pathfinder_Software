\doxysubsection{Detailed Description}
Container for rgb tau bitmap. 

The \char`\"{}width\char`\"{} and \char`\"{}height\char`\"{} paramter inform about the size of the bitmap. Every pixel value is stored in the data section of this class as 24 bit value. The 24 bits per pixel contain 3 RGB channels. Each channel value is 8 bit and all channels contain the same value. The original 14 bit values of the \mbox{\hyperlink{class_tau_raw_bitmap}{Tau\+Raw\+Bitmap}} pixels are scaled down to 8 bit resolution.
