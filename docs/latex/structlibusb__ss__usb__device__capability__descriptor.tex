\doxysubsection{Detailed Description}
A structure representing the Super\+Speed USB Device Capability descriptor This descriptor is documented in section 9.\+6.\+2.\+2 of the USB 3.\+0 specification. All multiple-\/byte fields are represented in host-\/endian format.
